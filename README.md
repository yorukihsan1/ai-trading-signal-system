# AI Trading Signal & Pattern Detection System

This project detects technical patterns in financial charts and generates trading signals (BUY/SELL/HOLD) using AI.

## Features
- Pattern Detection (CNN / YOLO)
- Signal Generation Engine
- Confidence Score
- Streamlit UI

## Project Structure
- data/: dataset
- models/: trained models
- src/: core logic
- app/: user interface

## Run
```bash
pip install -r requirements.txt
python main.py
